# Orivon Minds 2.0  
Versión estática mejorada de [orivonminds.com](https://orivonminds.com) lista para GitHub Pages.

## Uso
1. Haz fork o clona el repo.  
2. Edita `index.html` y cambia `tu@email.com` en el formulario.  
3. Activa GitHub Pages → Settings → Pages → Deploy from branch → main → /(root).  
4. ¡Listo! Tu web estará en `https://[usuario].github.io/[repo]/`.
